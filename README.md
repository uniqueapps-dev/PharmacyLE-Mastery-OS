# PharmacyLE-Mastery-OS
An adaptive, offline-first AI-powered study system for pharmacy licensing exams (PEBC, PSI, KAPS) designed to eliminate guesswork and enforce daily execution.
PharmacyLE Mastery OS

PharmacyLE Mastery OS is a high-performance, adaptive study system designed to help pharmacists prepare for international pharmacy licensing exams such as PEBC (Canada), PSI (Ireland), and KAPS (Australia).

This is not just a study app — it is a structured execution system built to remove reliance on motivation and replace it with daily guided action, performance tracking, and continuous adaptation.

Core Philosophy

Success should not depend on motivation.

This system is designed to:

- Eliminate decision fatigue (no guessing what to study)
- Enforce daily execution through structured tasks
- Identify and target weak areas automatically
- Adapt continuously based on performance
- Build exam readiness through repetition and simulation

Key Features

- 📚 Curriculum mapping across PEBC, PSI, and KAPS
- 🎯 Daily “mission-based” study system
- 🧠 Adaptive learning engine (focus on weak areas)
- 🧪 Practice system with MCQs, timed tests, and mock exams
- 🩺 OSCE simulation with structured feedback
- 🔁 Spaced repetition for long-term retention
- 📊 Performance tracking and weekly progress reports
- 🎧 Audio learning for passive study
- 📄 Daily study packet generation
- 💾 Offline-first design (works without constant internet)

Tech Stack

- Single-page application (SPA)
- Vanilla JavaScript (no frameworks)
- LocalStorage / IndexedDB for persistence
- Fully contained in a single HTML file

Vision

This project started as a personal system to prepare for international pharmacy licensing exams, but is being designed with the potential to evolve into a scalable learning platform (PharmBridge Project) for pharmacists seeking global opportunities.

Status

🚧 In active development — initial prototype being built with AI-assisted architecture.

Author
